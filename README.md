# Anne studio

Sitio web de una sola página (landing) para un estudio de maquillaje profesional. Presenta los servicios, la propuesta de la marca y los datos de contacto, con un diseño cálido orientado a belleza y bienestar.

🔗 **Demo en vivo:** https://romherre.github.io/anne-studio/

## Sobre el proyecto

Anne studio ofrece maquillaje social, para eventos (15 años, bodas, damas de honor), clases de automaquillaje, peinado y manicura. Incluye también un servicio de **maquillaje terapéutico**, pensado para acompañar a personas que atraviesan una enfermedad o un tratamiento y quieren sentirse mejor con su imagen — un servicio con un fin social, no solo estético.

## Secciones

- **Hero** — presentación de la marca
- **Servicios** — grilla con los 9 servicios ofrecidos, precios y detalle de cada uno
- **Quiénes somos** — propuesta de valor y estadísticas del estudio
- **Proceso** — pasos de la experiencia, de la consulta al resultado final
- **Contacto** — WhatsApp, Instagram y ubicación

## Tecnologías

- **HTML5** y **CSS3** puro (sin frameworks ni librerías)
- Tipografías de **Google Fonts**: Cormorant Garamond (títulos) y Jost (cuerpo de texto)
- Diseño responsive con CSS Grid y Flexbox
- Sin dependencias externas ni build step — es un único archivo `index.html`, fácil de editar y desplegar

## Estructura

```
Anne/
└── index.html   → contiene HTML, CSS y la lógica mínima del sitio
```

## Cómo verlo localmente

Solo hace falta abrir `index.html` en cualquier navegador, no requiere instalación ni servidor.

## Deploy

Publicado con **GitHub Pages** directamente desde la rama `main` del repositorio.
